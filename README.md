# Proyecto Final Automation Testing – Stefania Ocampo

##  Descripción
Framework de automatización de pruebas desarrollado en Python utilizando Pytest.
Incluye pruebas de UI con Selenium WebDriver y pruebas de API con Requests,
aplicando el patrón Page Object Model (POM).

##  Tecnologías
- Python 3
- Pytest
- Selenium WebDriver
- Requests
- pytest-html
- Faker
- Git / GitHub
